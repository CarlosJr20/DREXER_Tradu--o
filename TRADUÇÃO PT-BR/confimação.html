Confirmação para Jovens

O Programa de Educação Religiosa para Jovens está disponível para aqueles que precisam de preparação sacramental para a Confirmação.

Confirmação para adultos

O RCIA (Rito de Iniciação Cristã de Adultos) está disponível para todos os interessados em se tornar católicos e para católicos adultos que nunca foram confirmados. As sessões regulares são realizadas todas as segundas-feiras (setembro a Páscoa), até o recebimento dos sacramentos.
